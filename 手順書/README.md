## 路线表（一步一工具，Blender 为枢纽）

| 步 | 工具（主） | 谁主导 | 自检闸门（来自模块 selfCheck） |
|---|---|---|---|
| 01 概念设定 | Gemini / Nano Banana Pro | 我驱动浏览器 | 黑剪影还认得出 / 正侧背同一人 / 无强阴影透视花背景 |
| 02 3D 模型生成 | Tripo（备选 Meshy/混元） | 我驱动浏览器 | 剪影对齐 / 背侧面合理 / 比例原点法线朝向正常 |
| 03 雕刻 | Blender 4.5（雕刻） | 你手工 | 黑剪影认得出 / 每个角度都站得住 / 干净 A-pose |
| 04 重拓扑 | Blender QuadriFlow + 手修 | 我首过/你修关键环线 | 关节有环线脸有眼环嘴环 / 低模贴高模面数达标 / 体检干净 |
| 05 UV 展开 | Blender | 我自动/你调缝线 | 棋盘格密度一致 / 缝线藏好 / 岛不重叠留边距 |
| 06 纹理烘焙 | Blender Cycles（S2A） | 我烘焙/你修瑕疵 | 法线像高模无黑斑 / 切线空间三角化一致 / AO曲率ID 齐 |
| 07 上色 + 材质 | Meshy 底色 → Substance 精修 | 我驱动/你精修 | 材质读得对 / 底色平无光影 / 按引擎预设导出 |
| 08 绑定 + 动画 | Mixamo → Blender 权重修 | 我驱动/你修权重 | 关节弯到极限不塌 / 骨骼命名对齐 / 动画能播 |
| 09 导出到引擎 | Blender 导出 → Unity 6 | 我导出/你验收 | 比例朝向对 / 材质光照正确 / 动画能播能录转盘 |
